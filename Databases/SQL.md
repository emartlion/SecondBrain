# SQL

**Tags**:: #databases/sql
**Links**:: 

*SQL*, *S*tructured *Q*uery *L*anguaje, es un lenguaje usado para el manejo de datos, específicamente en sistemas de [[Bases de datos relacionales|bases de datos relacionales]]. Las bases de datos no relacionales se conocen como bases de datos ==NoSQL==.

Este lenguaje fue introducido en 1970 y permite al usuario realizar operaciones como *querying*, insertar, eliminar y actualizar datos de forma eficiente.

### Sabores
SQL se basa en los estándares ISO y ANSI. Sin embargo, con el paso de los años las empresas han desarrollado sus propias implementaciones de SQL para satisfacer sus necesidades específicas. Dichas variaciones de SQL se conocen como sus *sabores*. Los sabores de SQL son como los distintos dialectos de un idioma (e.g. Español 🇲🇽 y Español 🇪🇸).

Hay sabores comerciales como:

- Microsoft SQL Server
- MSSQL

Y sabores Open Source como:

- PostgreSQL
- MySQL
- SQLite

>[!note] Nota
>El orden de ejecución de SQL **NO** es el mismo que el de escritura. Siempre se ejecuta el comando `FROM` antes que cualquier otro, aunque no sea la primera línea.
