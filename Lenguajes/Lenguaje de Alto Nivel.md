# Lenguajes de Alto Nivel

**Tags**:: #lenguajes #lenguajes/general 
**Links**:: [[Python]], [[R]], [[Lenguaje Natural vs Máquina]]

Los lenguajes de alto nivel están diseñados para que su sintaxis sea fácil de entender y de producir por seres humanos. Son la reconciliación entre lo rudimentario del lenguaje máquina y la facilidad del lenguaje natural.

Se abstraen los detalles del hardware para alejarse del código máquina y de las instrucciones del procesador. Usualmente tienen un manejo automático de memoria y son portables, lo que los haces muy productivos en cuanto a producción y ejecución de código.

Algunos lenguajes de alto nivel son [[Python]], [[R]], JavaScript, Ruby, etc.

El contraste entre los lenguajes de alto, medio y bajo nivel para sumar 3 más 5 es evidente.

- Python (alto nivel)
```python
resultado = 3 + 5
```

- C (medio)
```C
int a = 3;
int b = 5;
int resultado = a + b;
```

- Assembly (bajo)
```asm
MOV AX, 5
ADD AX, 3
MOV resultado, AX
```

A los programas escritos en un lenguaje de alto nivel se les conoce como *source code*. Los archivos que contienen el source code son llamados *source files*.
