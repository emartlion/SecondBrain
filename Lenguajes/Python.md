# Python

**Tags**:: #lenguajes/python
**Links**:: [[Tipado Dinámico]],  [[Lenguaje Interpretado]]

---
[Python](https://www.python.org/) es un lenguaje de programación de alto nivel creado por Guido van Rossum. Este lenguaje está enfocado en la facilidad de la lectura de su código.

Es usado ampliamente en la comunidad de **Machine Learning** debido a diversos factores, técnicos y no tanto:

### No Técnicos
+ Mantenimiento permanente y  no centralizado.
	+ Al instalarlo ya tenemos una versión completa del lenguaje guardada localmente.
+ Tiene una alta compatibilidad con diversos tipos de datos (TXT, CSV, Excel, JSON, etc)
+ Tiene una alta conectividad con diversas fuentes de datos a través de APIs (Yahoo Finance, Google, Facebook, Instagram[^1]).
+ Versatilidad para atacar problemas de diversas áreas.
+ Es un lenguaje **Open Source**, cualquiera puede descargarlo y modificar su código. Es libre (*free as in speech*).
	+ Como consecuencia, recibe mejoras y auditorías constantes por parte de la comunidad.
+ Es gratis de usar (*free as in beer*).
+ Multiplataforma (Windows, macOS, Linux).
+ Es uno de los estándares en la industria.

[^1]: La accesibilidad de las APIs de Meta y X van de medianamente a altamente limitadas.

### Técnicas
- Es un [[Lenguaje Interpretado]] y no compilado.
- Es un [[Lenguaje de Alto Nivel]], con una sintaxis fácil de escribir y de leer.
- Orientado a la exploración interactiva (Read-Eval-Print Loop (*REPL*), *notebooks*).
	- Es ideal para prototipado rápido, análisis exploratorio de datos (*EDA*), fácil de enseñar.
- Enfoque moderno: Ciencia de Datos + Producción.
	- Data pipelines, APIs, MLOps, integración con la nube (AWS, GCP, Azure).
- [[Tipado Dinámico]].
- Las funciones son objetos nativos del lenguaje (*first-class citizens*).
- Se pueden pasar funciones como argumentos, retornarlas o guardarlas en variables.
- Soporta varios paradigmas de Programación:
	- [[Paradigma Imperativo o Procedural]]
	- Paradigma Funcional
	- Orientado a Objetos
