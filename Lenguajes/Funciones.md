# Funciones en Python

**Tags**:: #lenguajes/python #lenguajes/python/funciones 
**Links**:: [[Python]]

Las funciones en Python son primas de las funciones matemáticas que conocemos. Éstas funciones son más flexibles y pueden tener más contenido que las funciones matemáticas clásicas. En este contexto, una función puede:

- **Causar algún efecto:** enviar texto a la terminal, crear un archivo, dibujar una imagen, reproducir un sonido, etc.
- **Evaluar algún valor:** la raíz cuadrada de un valor o la longitud de un texto dado y **devolverlo como el resultado de la función**.

En general, las funciones de Python pueden hacer ambas cosas a la vez.


## ¿De dónde vienen las funciones?

Pueden provenir de diferentes fuentes:

### 1. Del mismo Python
La función ``print`` es un ejemplo de estas funciones. Dicha función viene incluida con Python y su ambiente (*built-in*). No se necesita hacer nada extra para poder usarla.

### 2. De módulos
Python tiene add-ons llamados *módulos*; algunos de ellos vienen en Python, otros necesitan una instalación por separado. Cual sea el caso, estos módulos necesitan estar conectados explícitamente con el código.

### 3. De nuestro código
Podemos escribirlas nosotros mismos. Podemos crear y usar tantas funciones como queramos y necesitemos en el código. El nombre de las funciones debe destacar (ser auto-evidente).
