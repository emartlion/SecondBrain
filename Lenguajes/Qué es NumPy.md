# ¿Qué es NumPy?

**Tags**:: #lenguajes/python/numpy 
**Links**:: [[Python]],

[*NumPy*](https://numpy.org/), abreviación de Numeric Python, es una librería ([[Módulo]]) de dicho lenguaje que añade características matemáticas más avanzada, como una nueva clase de objeto llamado *array* (una lista, pero más fácil de operar) y una gran colección de funciones matemáticas de alto nivel.

Este módulo es una evolución de la librería  *Numeric*, desarrollada por [Jim Hugunin](https://en.wikipedia.org/wiki/Jim_Hugunin). Posteriormente, [Travis Oliphant](https://en.wikipedia.org/wiki/Travis_Oliphant) añadió características de Numarray (una librería rival) a Numeric, creando así NumPy.

### Instalación
Este módulo se puede instalar escribiendo el siguiente comando desde la terminal:

```powershell
pip install numpy
```
