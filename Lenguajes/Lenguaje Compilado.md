# Lenguaje Compilado

**Tags**:: #lenguajes #lenguajes/general 
**Links**:: [[Lenguaje Interpretado]], [[Lenguaje Natural vs Máquina]]

Los lenguajes compilados son aquellos cuyo *source code* necesita ser traducido a lenguaje máquina a través de una herramienta llamada **compilador**.

El source code es "traducido" una vez y se genera un archivo ejecutable (.exe en Windows) escrito en código máquina, listo para distribuido y ejecutado sin problemas.

>[!note] Nota
>Cada vez que se realice una modificación al source code, el programa necesitará compilarse nuevamente.

Algunos lenguajes compilados son: C, C++, Rust, Go.
