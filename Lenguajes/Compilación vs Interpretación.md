# Compilación vs Interpretación: Ventajas y Desventajas

**Tags**:: #lenguajes #lenguajes/general 
**Links**:: [[Lenguaje Compilado]], [[Lenguaje Interpretado]]

### ¿Qué modelo de ejecución es mejor?
No hay una respuesta clara. Ambos tienen sus ventajas y sus desventajas.

|             | Compilación                                                                                                                              | Interpretación                                                                                                                                                                                            |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Ventajas    | ✅ La ejecución del código suele ser más rápida.                                                                                          | ✅ Puedes correr el código tan pronto como lo termines.                                                                                                                                                    |
|             | ✅ Sólo el usuario necesita tener el compilador. El usuario final puede correr el código sin él.                                          | ✅ El código se encuentra en un leguaje de programación - significa que puede correr en computadoras que tengan distinto lenguaje de máquina. No hay necesidad de compilar para cada tipo de arquitectura. |
|             | ✅ El ejecutable está escrito en lenguaje máquina - como es muy difícil de entender, tus secretos y trucos de programación están a salvo. |                                                                                                                                                                                                           |
|             |                                                                                                                                          |                                                                                                                                                                                                           |
| Desventajas | ❌ El mismo proceso de compilación puede llegar a ser tardado - no puedes ejecutar tu código inmediatamente después de terminarlo.        | ❌ Tu código compartirá el poder de cómputo de la computadora con el intérprete, por lo que no podrá usar los recursos al máximo.                                                                          |
|             | ❌ Necesitas tener tantos compiladores como el número de distintas plataformas que quieras alcanzar con tu software.                      | ❌ Tanto tú como el usuario final necesitan tener el intérprete para poder ejecutar el código.                                                                                                             |
