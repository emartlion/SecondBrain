# Booleanos

**Tags**:: #lenguajes/python/literales 
**Links**:: [[Python]], [[Valores Literales]]

Este tipo de valor se usa para representar una idea bastante abstracta: la veracidad.

Cada vez que le preguntamos a [[Python]] si un número es mayor que otro, se genera un tipo específico de dato: un *booleano*, `bool`.

El nombre viene de [George Boole](https://en.wikipedia.org/wiki/George_Boole) (1815-1864), el autor de *The Laws of Thought*, que contiene la definición del álgebra booleana, un álgebra que sólo hace uso de dos valores: `True` y `False`, `0` y `1`.

Cuando un programa hace preguntas, Python lo ejecuta y provee respuestas. El programa debe poder reaccionar de acuerdo a las respuestas recibidas. Para las computadoras sólo hay dos respuestas:

- Sí, esto es cierto `True`;
- No, esto es falso `False`.
